## Productivity is workflow design

The best productivity tools reduce context switching and help you decide what matters next. They should support behavior, not become a hobby.

## Core stack

- Calendar for time commitments
- Task manager for next actions
- Notes app for thinking and reference
- Automation tool for repeated handoffs
- AI assistant for drafting and synthesis

## Selection rule

Choose fewer tools and connect them well. A simple system you trust beats a beautiful system you avoid.
