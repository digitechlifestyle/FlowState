# DigiTech Lifestyle — Design Context

## Design Context

### Users
UK-based crypto/AI enthusiasts, ages 25–50. Mix of beginners learning about Bitcoin and DeFi alongside experienced traders seeking honest reviews. Browsing on desktop evenings after work, mobile on commute. Context: low-trust environment — they've been burned by hype before. They want clarity, not sales.

### Brand Personality
**Words:** sovereign, editorial, earned-trust
Premium independent publication. Not a VC-backed exchange. Joe Robertson is a real person who has been in crypto since 2017 and writes plain English. The brand should feel like a Bloomberg Opinion by someone who actually holds the assets they write about.

### Aesthetic Direction
Dark editorial fintech. Black background (near pure, slight warm tint). Gold/amber (#F59E0B) as the single accent — like a Financial Times masthead in Web3. Purple (#7C3AED) as a secondary neon for tech/AI content only. 3D floating objects, orbital ring effects, neon glow — used on feature images, NOT on UI chrome.

**Anti-references:** No cyan-on-dark startup vibes. No "web3 gradient everything." No glassmorphism. No border-left accent stripes.

**Reference:** The two brand images uploaded by Joe — Automation Playbook (portrait, purple orbital rings, 3D clipboard) and Resource Hub (landscape, circuit lines, laptop mockup) — define the image aesthetic precisely.

### Font System
- Display/headings: `Sora` (already loaded) — bold, geometric, confident
- Body: `Manrope` (already loaded) — clean, readable, warm

### Design Principles
1. **Trust through restraint** — every decorative element must earn its place; if in doubt, remove it
2. **Editorial hierarchy** — clear information cascade, big heading → sub-head → body → meta
3. **Gold is precious** — use amber accent sparingly; it should stand out when used
4. **Images carry the atmosphere** — the brand images do the heavy lifting; UI chrome stays minimal
5. **Performance first** — lazy-load everything, WebP, no animation that delays content paint
