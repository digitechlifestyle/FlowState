📋 What This Playbook Covers

This playbook shows you how to automate your most repetitive business tasks without writing a single line of code. It covers the four core automation areas for digital entrepreneurs: content, email, admin, and customer touchpoints.

🧠 The Automation Framework

Every automation has three parts. Understanding these makes any new workflow easy to design.

1. TRIGGER — What starts it?

A trigger is the event that kicks off the automation. Examples: form submission, email received, new blog post published, calendar date reached, file added to folder.

2. FILTER — When should it run?

Filters let you add conditions. Examples: only run if the email contains "invoice", only run if the subscriber tag equals "new-lead", only run between 9am–5pm.

3. ACTION — What does it do?

The action is what happens automatically. Examples: send email, create task, add row to spreadsheet, post to Slack, schedule social post, tag subscriber.

1

Track publish dates in a Google Sheet

Column A: article URL. Column B: publish date. Column C: last-refreshed date.

2

Set a Zapier "Schedule" trigger

Every 1st of the month, Zapier reads the sheet and finds rows where Column B is more than 6 months ago.

3

Action: create Notion task

"Refresh article: [URL]" appears in your content backlog automatically. No manual tracking needed.

🗂️ Workflow 3 — Admin Automation

Invoice Reminders

Set up in your accounting tool (FreeAgent, QuickBooks, Wave): automatic email 7 days before due, 1 day before due, and day after overdue. Takes 10 minutes to configure — saves awkward chasing forever.

Weekly Report Email

Use Zapier + Google Analytics + Sheets: every Monday at 8am, pull last week's traffic, email list size, and top page. Auto-email yourself a summary. No manual logging.

File Naming + Backup

Use Hazel (Mac) or Zapier: when a file lands in your Downloads folder matching a pattern (e.g. "invoice"), auto-rename it and move to the right folder. Never hunt for files again.

Subscription Audit Reminder

Set a recurring Google Calendar event on the 1st of each month: "Review subscriptions". List all tools in a Notion table with monthly cost. Cancel unused ones. Most people save £50–£200/month.

📱 Workflow 4 — Social Media Automation

Weekly Batch + Schedule System

1

Monday: batch content (90 min)

Write 5 posts, 1 thread, 2 Instagram captions. Save in a folder or Notion page.

2

Monday: schedule everything

Upload all posts to Buffer or Postiz. Set optimal times (LinkedIn: Tue–Thu 8am/5pm. X: varied. Instagram: 11am/7pm).

3

Reply to comments. Like and comment on 5 posts from target accounts. Don't scroll — in and out in 15 min.

4

Friday: review top performer

Note the post with highest engagement. What made it work? Repeat the pattern next week.

📈 When to Scale Automation

Don't try to automate everything at once. Follow this progression:

Month 1 — Foundation
  • Lead magnet delivery automation live
  • Welcome email sequence set up (3 emails)
  • Social content batched and scheduled weekly
  • Content tracker spreadsheet created and used
Month 2 — Expand
  • Publish → promote pipeline connected
  • Invoice reminders automated
  • Weekly analytics email set up
  • Re-engagement sequence for inactive subscribers added
Month 3+ — Refine
  • Review all automations: any breaking? Any no longer needed?
  • Add new automations only where manual time is still high
  • Document each automation as an SOP for VA handoff
  • Calculate time saved monthly — use it for higher-leverage work
